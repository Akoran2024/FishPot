5.2 Descripción de Ficheros Clave

5.2.1 Controladores y Lógica de Servidor (PHP)

Lógica app/Http/Controllers/TideController.php Encargado de la comunicación con WorldTides API. Procesa las coordenadas de Lanzarote y formatea los datos de pleamar/bajamar para el frontend.
Lógica app/Http/Controllers/OrderController.php Gestiona el flujo transaccional. Valida el stock antes de confirmar una compra y genera el evento de facturación.
Seguridad app/Http/Middleware/AdminMiddleware.php Verifica el rol de administrador en las peticiones HTTP. Es la barrera de seguridad para el panel de gestión.

5.2.2 Interfaz y Experiencia de Usuario (Vue.js)

UI resources/js/views/Shop.vue Página principal de comercio electrónico. Implementa filtros reactivos por categoría y búsqueda de productos en tiempo real.
Estado resources/js/stores/cart.js Almacén centralizado del carrito de compras mediante Pinia. Persiste los artículos seleccionados durante la navegación del usuario.
UI resources/js/components/Toast.vue Componente de notificación global. Muestra avisos visuales (éxito/error) tras acciones como añadir al carrito o login.
