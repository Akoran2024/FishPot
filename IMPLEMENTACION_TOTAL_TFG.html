5.2 Catálogo Exhaustivo de Ficheros

5.2.1 Controladores de Aplicación (app/Http/Controllers)

ControllerClientController.php

Gestión integral de la entidad Cliente. Maneja el listado y los datos de contacto/envío.

ControllerOrderController.php

Lógica central de pedidos: creación, validación de stock, cálculo de totales y actualización de estados (Aceptado/Enviado).

ControllerProductController.php

Controlador para el inventario de la tienda. Gestiona categorías, precios y carga de imágenes de productos.

ControllerProfileController.php

Maneja la edición de datos del usuario autenticado y la baja de cuentas.

ControllerTideController.php

Integración con API WorldTides. Formatea datos astronómicos para el visor de mareas de Lanzarote.

ControllerAdmin/UserController.php

Panel administrativo de usuarios. Permite la gestión de roles y permisos de acceso.

5.2.2 Controladores de Autenticación (app/Http/Controllers/Auth)

AuthAuthenticatedSessionController.php

Gestión de inicio y cierre de sesión (Login/Logout).

AuthRegisteredUserController.php

Lógica de creación de nuevos usuarios en el sistema.

AuthPasswordController.php

Actualización de contraseñas de usuarios autenticados.

AuthNewPasswordController.php

Procesamiento de restablecimiento de contraseña olvidada.

AuthPasswordResetLinkController.php

Envío de enlaces de recuperación de cuenta por email.

AuthVerifyEmailController.php

Lógica de verificación de dirección de correo electrónico.

5.2.3 Modelos de Datos (app/Models)

ModelUser.php

Modelo principal. Gestiona autenticación, roles (admin/user) y relación con pedidos.

ModelProduct.php

Define atributos de artículos: nombre, descripción, precio, stock y categoría.

ModelOrder.php

Cabecera de pedidos. Almacena totales, estados y datos de facturación.

ModelOrderItem.php

Relación muchos-a-muchos entre pedidos y productos con cantidades y precios históricos.

ModelClient.php

Extensión de datos de contacto para envíos y CRM.

5.2.4 Validaciones de Entrada (app/Http/Requests)

RequestOrderStoreRequest.php

Reglas para validar que un pedido tenga productos válidos y stock suficiente.

RequestProductStoreRequest.php

Validación de campos obligatorios al crear nuevos productos (precio positivo, imágenes válidas).

RequestProfileUpdateRequest.php

Asegura que los cambios en el perfil (como el email) sean únicos en la DB.

RequestAuth/LoginRequest.php

Control de intentos de acceso y validación de credenciales.

5.2.5 Migraciones de Base de Datos (database/migrations)

MigraciónDescripción de la Operación
create_users_tableCrea la tabla base de usuarios y gestión de tokens.
add_role_to_users_tableInyecta el sistema de permisos mediante la columna 'role'.
create_products_tableDefine el catálogo de inventario.
create_orders_tableCrea las tablas de pedidos y líneas de detalle.
add_shipping_details_to_ordersAñade campos para dirección física de entrega.
add_payment_method_to_ordersIntegra la gestión del modo de pago seleccionado.
create_clients_tableEstructura para la base de datos de clientes registrados.

5.2.6 Frontend: Vistas y Páginas (resources/js/views)

ViewHome.vue

Landing page con banners dinámicos y acceso rápido.

ViewShop.vue

Catálogo de productos con filtrado por categorías.

ViewTides.vue

Interfaz visual del visor de mareas de Lanzarote.

ViewCart.vue

Resumen de carrito y pasarela de finalización de compra.

ViewMyOrders.vue

Historial de compras personal del cliente.

ViewSpecies.vue

Enciclopedia visual de peces locales de las Canarias.

Viewadmin/Orders.vue

Panel de gestión y despacho de pedidos para administración.

ViewUsers.vue

CRUD de usuarios del sistema.

5.2.7 Frontend: Gestión de Estado (resources/js/stores)

Storecart.js

Persistencia del carrito en LocalStorage y reactividad en la UI.

Storeauth.js

Gestión centralizada del usuario autenticado y sus permisos.

Storetoast.js

Cola de notificaciones para eventos asíncronos.

5.2.8 Sistema de Rutas (routes)

Routesweb.php

Define los endpoints de la API, las rutas administrativas protegidas y el punto de entrada de la SPA.

Routesauth.php

Rutas exclusivas del sistema de autenticación (Breeze/Fortify).

5.3 Infraestructura de Despliegue

InfraDockerfile

Configuración del entorno PHP-FPM con extensiones necesarias (GD, PDO, ZIP).

Infradocker-compose.yml

Orquestación de servicios: App (PHP), Web (Nginx), DB (MySQL) y Mail (Mailpit).

Infravite.config.js

Configuración del empaquetador para la compilación de activos de Vue y Tailwind.
