Documentación técnica para el Trabajo de Fin de Grado

1. Arquitectura del Sistema

FishPot es una aplicación robusta construida con una arquitectura de desacoplamiento total entre el cliente y el servidor. Se utiliza Laravel 12 para la gestión de la lógica de negocio y persistencia de datos, y Vue.js 3 para una interfaz de usuario reactiva y fluida.

3. Descripción Detallada de Archivos

3.1 Núcleo de la Aplicación (Directorio app/)

Archivo/Carpeta Descripción
app/Http/Controllers/StripeController.php Punto clave de la lógica comercial. Gestiona la creación de sesiones de pago y procesa el éxito de los pedidos, actualizando el stock automáticamente.
app/Http/Controllers/TideController.php Controlador especializado que interactúa con APIs externas para obtener mareas de Lanzarote, con lógica de respaldo si falla el servicio.
app/Models/Order.php Representa la entidad Pedido, relacionando usuarios con productos y detalles de envío.
app/Http/Middleware/AdminMiddleware.php Filtro de seguridad que valida privilegios de administrador para proteger el acceso al panel CRUD.

3.2 Interfaz de Usuario (Directorio resources/js/)

Archivo/Carpeta Descripción
views/Shop.vue Página principal de comercio electrónico. Implementa filtros reactivos por categoría de productos.
views/Cart.vue Gestión del proceso de compra. Permite modificar cantidades y calcular gastos de envío en tiempo real.
stores/cart.js Almacén central de datos del carrito (Pinia). Mantiene la persistencia incluso si el usuario refresca la página.
views/Tides.vue Componente visual que muestra gráficos e información sobre las mareas para la planificación de la pesca.

3.3 Configuración y Base de Datos

Archivo/Carpeta Descripción
database/migrations/ Colección de archivos que definen la estructura SQL. Incluye campos para imágenes, precios y detalles de pago.
database/seeders/ProductSeeder.php Guion que carga el catálogo real de productos y especies de Lanzarote en el sistema.
public/imagenes/ Repositorio de imágenes de alta calidad para cada especie y producto (mero, vieja, sargo, etc.).
docker-compose.yml Orquestación de servicios: Servidor PHP 8.2, Base de Datos MySQL y Servidor Web Nginx.
