Jerarquía Visual de Directorios y Funcionalidad

  • FishPot/ Directorio raíz del proyecto.
    • app/ Núcleo del backend (Lógica de negocio, Modelos y Controladores).
      • Http/ Gestión de la capa de transporte y peticiones web.
        • Controllers/ Controladores que procesan la lógica de las solicitudes.
        • Middleware/ Filtros de seguridad y verificación de roles.
        • Requests/ Clases de validación de datos para formularios.
      • Models/ Definición de entidades y relaciones de la base de datos.
      • Mail/ Lógica para la generación y envío de correos electrónicos.
    • config/ Archivos de configuración global de servicios y del framework.
    • database/ Gestión de la estructura y población de datos.
      • migrations/ Scripts de creación y modificación de tablas SQL.
      • seeders/ Carga de datos maestros y catálogo inicial.
    • public/ Archivos públicos: Punto de entrada, imágenes y assets compilados.
      • imagenes/ Repositorio de fotos de especies y productos de pesca.
    • resources/ Código fuente del frontend y plantillas de servidor.
      • js/ Lógica de Vue.js: Vistas, componentes y gestión de estado Pinia.
        • views/Páginas de la Single Page Application.
        • stores/Gestión de estado global (Carrito, Auth).
      • views/ Plantillas Blade para la carga de la SPA y correos/PDFs.
    • routes/ Definición de endpoints API y rutas de navegación web.
