Jerarquía de Directorios (Source Code)

# Directorio Raíz del Proyecto

app/ # Lógica de negocio (PHP)

├── Http/Controllers/ // Stripe, Tide, Product, Order...

├── Models/ // User, Product, Order, Item...

└── Mail/ // Generación de facturas PDF

database/ # Persistencia

├── migrations/ // Definición de esquema SQL

└── seeders/ // Datos de especies y productos

resources/js/ # Frontend (Vue 3)

├── views/ // Home, Shop, Cart, Tides, MyOrders...

├── stores/ // Pinia (Cart & Auth States)

└── components/ // Componentes UI Reutilizables

public/ # Estáticos

└── imagenes/ // Assets de especies y artículos

Análisis por Componentes

Capa de Servicios (Backend)

PHP 8.2 + Framework Laravel

Controladores de Dominio

Gestión de la lógica transaccional. `StripeController` coordina los pagos externos y `TideController` abstrae la complejidad de la API de mareas.

Seguridad y Middleware

Protección de rutas sensibles mediante `AdminMiddleware`, verificando roles en tiempo real antes de conceder acceso al panel.

Capa de Presentación (Frontend)

Vue.js 3 + SPA Architecture

Reactividad con Pinia

El estado del carrito (`cart.js`) y la sesión del usuario se gestionan de forma centralizada, garantizando sincronización en todas las vistas.

Navegación Dinámica

Uso de `Vue Router` para transiciones suaves. El sistema carga componentes bajo demanda para optimizar el rendimiento inicial.
