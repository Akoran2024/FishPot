Descripción de Ficheros del Proyecto FishPot

Este documento detalla la función de los archivos principales del proyecto, organizados por su responsabilidad dentro de la arquitectura MVC y la SPA.

1. Backend y Lógica de Negocio (Laravel)

app/Http/Controllers/StripeController.php Backend
Gestiona la integración con la pasarela de pagos Stripe. Crea las sesiones de checkout y procesa la respuesta del servidor tras un pago exitoso, creando el pedido en la base de datos.
app/Http/Controllers/TideController.php Backend
Controlador encargado de la lógica de mareas. Realiza peticiones a la API externa WorldTides y formatea los datos para su visualización en el frontend.
app/Http/Controllers/ProductController.php Backend
Gestiona el catálogo de productos, incluyendo la búsqueda, filtrado por categorías y la administración de stock.
app/Models/Order.php Backend
Modelo Eloquent que representa un pedido. Define las relaciones con los usuarios y los ítems del pedido (OrderItems).
app/Http/Middleware/AdminMiddleware.php Backend
Filtro de seguridad que verifica si el usuario autenticado tiene rol de administrador antes de permitir el acceso a rutas restringidas.
app/Mail/InvoiceMail.php Backend
Define el formato y contenido del correo electrónico de factura que se envía automáticamente al cliente tras una compra.

2. Frontend e Interfaz (Vue.js)

resources/js/views/Shop.vue Frontend
Componente principal de la tienda. Muestra el catálogo de productos con una interfaz reactiva que permite filtrar sin recargar la página.
resources/js/views/Cart.vue Frontend
Interfaz del carrito de compras. Permite al usuario revisar sus productos, ajustar cantidades y proceder al pago con Stripe.
resources/js/stores/cart.js Frontend
Estado global del carrito usando Pinia. Gestiona la persistencia de los productos seleccionados en el navegador del usuario.
resources/js/views/Tides.vue Frontend
Muestra la información de mareas mediante gráficos y tablas, facilitando la planificación para los pescadores.
resources/js/router/index.js Frontend
Configuración de Vue Router. Define todas las rutas de la SPA y gestiona la navegación del lado del cliente.

3. Configuración e Infraestructura

database/migrations/xxxx_create_products_table.php Database
Define la estructura de la tabla de productos, incluyendo campos para nombre, descripción, precio, stock e imagen.
routes/api.php Routes
Define los endpoints de la API REST que consume el frontend de Vue para obtener datos de productos, mareas y gestionar pedidos.
docker-compose.yml DevOps
Archivo de orquestación de Docker que levanta los servicios de PHP, Nginx y MySQL para garantizar un entorno de desarrollo uniforme.
vite.config.js Tooling
Configuración del empaquetador Vite para la compilación optimizada de los activos de Vue y CSS (Tailwind).
.env Environment
Variables de entorno críticas: credenciales de base de datos, claves de la API de Stripe y configuración de correo.
