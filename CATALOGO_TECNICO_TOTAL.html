Anexo: Catálogo Técnico Completo de Ficheros del Proyecto

A continuación se presenta el desglose absoluto de todos los archivos que componen el sistema FishPot, detallando su ubicación exacta y su responsabilidad técnica dentro de la aplicación.

1. Configuración de Raíz y Entorno

Ruta del FicheroDescripción Técnica
.editorconfigDefine estándares de codificación consistentes para diferentes editores e IDEs.
.envArchivo de configuración de variables de entorno (Base de datos, Stripe, Mail).
.env.examplePlantilla de configuración para replicar el entorno de desarrollo.
artisanInterfaz de línea de comandos de Laravel para tareas de mantenimiento y ejecución.
composer.jsonGestor de dependencias y configuración del proyecto PHP.
composer.lockRegistro exacto de las versiones de las librerías PHP instaladas.
docker-compose.ymlOrquestación de contenedores Docker (App, Web, DB).
DockerfileInstrucciones para la construcción de la imagen Docker de la aplicación.
package.jsonGestor de dependencias y scripts de Node.js (Vite, Vue, Tailwind).
package-lock.jsonBloqueo de versiones de las dependencias de Node.js.
phpunit.xmlConfiguración para la ejecución de pruebas automatizadas con PHPUnit.
postcss.config.jsConfiguración para el procesamiento de CSS con PostCSS.
tailwind.config.jsConfiguración del framework de estilos Tailwind CSS.
vite.config.jsConfiguración del empaquetador Vite para el frontend.

app/Models/OrderItem.phpModelo para el detalle de productos dentro de un pedido.

app/Models/User.phpModelo de la entidad Usuario (autenticación y roles).
app/Providers/AppServiceProvider.phpProveedor de servicios para la configuración global de la aplicación.
app/View/Components/GuestLayout.phpComponente de layout para vistas de usuarios no autenticados.
